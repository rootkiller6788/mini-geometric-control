# Prerequisite Tree: Feedback Linearization

## Before This Module
1. Linear Algebra: Vector spaces, matrices, rank, nullspace, eigenvalues
2. Calculus: Partial derivatives, gradients, Jacobians, Taylor series
3. Differential Equations: ODEs, phase portraits, equilibrium stability
4. Linear Control Theory: State-space models, controllability, observability
5. Differential Geometry (basic): Manifolds, tangent spaces, vector fields

## This Module
6. Lie derivatives and Lie brackets (foundational operations)
7. Frobenius theorem and integrability (geometric condition)
8. Relative degree and normal forms (structural properties)
9. Input-output feedback linearization (partial linearization)
10. Input-state feedback linearization (full linearization)
11. Zero dynamics and minimum phase analysis (internal stability)
12. MIMO feedback linearization (multi-input multi-output)

## After This Module
13. Backstepping control design
14. Sliding mode control (comparative methods)
15. Adaptive nonlinear control
16. Model predictive control (linearization-based MPC)
17. Flatness-based control (differential flatness)
18. Geometric optimal control (Pontryagin on manifolds)
19. Hamiltonian control (symplectic geometry, passivity)
20. Nonlinear observer design (dual of feedback linearization)

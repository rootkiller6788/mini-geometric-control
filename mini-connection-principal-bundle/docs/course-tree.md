# Course Dependency Tree: Connection on Principal Bundles

## Prerequisites (what this module depends on)

### Level 0: Mathematical Foundations
- Linear algebra: matrices, eigenvalues, determinants, orthogonality
- Calculus: differentiation, integration, differential equations
- Group theory: groups, homomorphisms, representations

### Level 1: Differential Geometry Foundations
- Manifolds: charts, atlases, tangent spaces, vector fields
- Fiber bundles: local trivializations, sections
- Lie groups: group manifold, left/right translations, exponential map

### Level 2: Intermediate Topics
- Riemannian geometry: metric, Levi-Civita connection, geodesics
- Lie algebras: structure constants, Killing form, adjoint representation
- Differential forms: exterior derivative, wedge product

## Dependents (what depends on this module)

### Direct Dependencies
- Gauge Theory: Yang-Mills, Chern-Simons, BF theory
- Lattice QCD: numerical simulation of strong interactions
- Geometric Quantization: prequantum line bundles
- Topological Insulators: Berry curvature, Chern numbers
- Geometric Control: optimal control on Lie groups

### Indirect Dependencies
- Quantum Chromodynamics (QCD)
- Standard Model of particle physics
- String Theory (Calabi-Yau bundles)
- Topological Quantum Computing (anyons, braiding)
- Quantum Hall Effect (Chern-Simons effective theory)

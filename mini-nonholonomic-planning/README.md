# Mini Nonholonomic Motion Planning

Nonholonomic motion planning on configuration manifolds.
Implements the complete pipeline from geometric analysis to
sampling-based planning for systems with non-integrable
velocity constraints.

## Core Concept

**Nonholonomic constraints** limit velocities, not positions.
A system with m < n controls can still reach any n-dimensional
configuration by exploiting Lie bracket motions — this is the
essence of nonholonomic motion planning.

```
Unicycle: q̇ = g₁(q)·v + g₂(q)·ω
  g₁ = (cos θ, sin θ, 0)^T   ← forward
  g₂ = (0, 0, 1)^T            ← rotate
  [g₁, g₂] = (-sin θ, cos θ, 0)^T  ← side-slip (NOT directly actuated!)
```

## Key Theorems

### Frobenius Theorem (1877)
> A distribution Δ is integrable (holonomic) ⇔ Δ is involutive (closed under Lie bracket)

```
Test: for all i, j: [g_i, g_j] ∈ span{g₁, ..., g_m}
→ If YES: constraints are HOLONOMIC (can be integrated)
→ If NO:  constraints are NONHOLONOMIC (genuine velocity constraint)
```

### Chow-Rashevskii Theorem (1938/1939)
> If Lie algebra generated by {g_i} spans T_qQ for all q, the system is controllable

```
LARC test: rank(span{g_i, [g_j, g_k], [g_l, [g_m, g_n]], ...}) = n
→ If rank = n: system is STLC (small-time locally controllable)
```

### Brockett's Necessary Condition (1983)
> Smooth time-invariant feedback ⇒ (q, u) → f(q, u) surjective near 0

```
For driftless systems: m ≥ n is necessary for smooth stabilization
→ Unicycle (n=3, m=2): CANNOT be smoothly stabilized!
→ Must use time-varying feedback or discontinuous control
```

## Planning Algorithms

### Sampling-Based
- **RRT**: Rapidly-exploring Random Tree with nonholonomic forward simulation
- **RRT-Connect**: Bidirectional variant for narrow passages
- **PRM**: Probabilistic Roadmap with nonholonomic edge connections
- **Lattice A***: Search on pre-computed motion primitive graph

### Geometric Steering
- **Sinusoidal**: Murray-Sastry (1993) — for chained-form systems
- **Nilpotent**: Lafferriere-Sussmann (1993) — Philip Hall coordinates
- **Random**: Forward simulation with randomized controls

### Path Optimization
- **Shortcut smoothing**: Iterative sub-path replacement
- **Gradient descent**: Minimize path length + control effort
